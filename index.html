Introdução

Bem-vindo ao nosso portifólio acadêmico, apresentando uma jornada de aprendizado, colaboração e inovação ao longo do segundo semestre. Este portifólio é um reflexo do compromisso e da dedicação dos membros deste grupo em explorar, desenvolver e implementar soluções tecnológicas significativas.
Durante este período, mergulhamos em uma série de projetos desafiadores, cada um representando uma faceta única do nosso crescimento acadêmico e profissional. Os quatro principais projetos destacados aqui - Projeto Intermediário e Final de Linguagem de Programação, Intercâmbio Brasil-China via Lark e Projeto Integrador I.

Projeto Intermediário de Linguagem de Programação

O Projeto Intermediário de Linguagem de Programação foi um software desenvolvido pelo grupo deste portifólio, o sistema consiste em código feito na Linguagem C, no qual o programa calcula a área geométrica de formas como: Quadrado, Retângulo, Triângulo e Trapêzio.

Projeto Final de Linguagem de Programação

O Projeto Final de Linguagem de Programação foi um software desenvolvido pelo grupo deste portifólio, o sistema consiste em um web-site que calcula o valor de Participação de Lucros e Resultados de uma empresa (PLR). Este programa foi desenvolvido em HTML, CSS e JavaScript.
